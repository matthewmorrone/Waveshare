# Waveform Standalone Boards

These folders are standalone PlatformIO projects that reuse shared code from `Shared` while enabling only the screen profile they need.

Current standalone projects:
- `flash Calculator`
- `flash Calendar`
- `flash Cube`
- `flash GPS`
- `flash IMU`
- `flash Launcher`
- `flash Locket`
- `flash Motion`

- `flash QR`

- `flash Settings`
- `flash Shared`
- `flash Sound`
- `flash Spectrum`
